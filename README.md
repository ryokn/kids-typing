# 🎹 たのしいタイピング

幼稚園児（4〜5歳）向けのWebタイピング練習アプリです。

**🌐 公開URL: https://ryokn.github.io/kids-typing/**

---

## 概要

ひらがなの単語が表示され、ローマ字でキーボード入力する練習ができます。
キーボードハイライトやヒント表示があるので、キーボードを覚えたての子供でも安心して遊べます。

---

## 機能

### ゲームプレイ
- **5段階のレベル**: やさしい〜チャンピオンまで
- **ひらがな＋ローマ字表示**: 入力済みの文字が色で変わる
- **キーボード表示**: 次に押すキーがハイライト
- **複数ローマ字対応**: `SHI` / `SI`、`CHI` / `TI`、`TSU` / `TU`、`FU` / `HU` など、どちらの入力方式でも正解

### 👹 おにモード
- キーボード表示・ヒントを非表示にした上級者向けモード
- 獲得XPが2倍

### ⏱ タイマー・記録
- プレイ時間を計測
- レベルごとにベストタイムを記録
- 記録更新時にボーナスXP

### ⭐ 経験値・ランクシステム
- 問題クリアごとにXP獲得
- ミスなしパーフェクトクリアはXP1.5倍
- 6段階のランク（ひよこ→こねこ→こぎつね→たぬき→らいおん→チャンピオン）

### 🎰 ガチャシステム
- ステージクリアでチケット獲得（Lv1〜3: 1枚 / Lv4〜5: 2枚）
- 5段階レアリティ（コモン〜レジェンド）のアニマルアイテム全23種
- コレクション画面でアイテムの収集状況を確認

### 🔊 サウンド・BGM
- レベルごとにBPMが変わるBGM
- キー入力音・正解音・クリア音
- ミュートボタンあり

## レベル詳細

| レベル | 難易度 | 問題数 | 内容 |
|--------|--------|--------|------|
| Lv.1 | やさしい | 5問 | 2文字の基本単語 |
| Lv.2 | ふつう | 6問 | 2〜3文字 |
| Lv.3 | むずかしい | 7問 | 3〜4文字・shi/chi/tsu含む |
| Lv.4 | もっとむずかしい | 8問 | 4〜5文字・拗音含む |
| Lv.5 | チャンピオン | 10問 | 長い単語・複雑な組み合わせ |

## データ保存

プレイデータはブラウザの `localStorage` に保存されます。

- 保存内容: XP・ランク・ベストタイム・チケット枚数・コレクション
- サーバーへの送信なし（完全ローカル）
- 別のブラウザ・端末との共有は不可

### データの確認・初期化

ブラウザの開発者ツール（F12）> Console で実行：

```js
// データを確認
JSON.parse(localStorage.getItem('kidsTypingData'))

// データを初期化
localStorage.removeItem('kidsTypingData')
```

## ローカルで実行

```bash
git clone https://github.com/ryokn/kids-typing.git
cd kids-typing
open index.html   # macOS
# または index.html をブラウザにドラッグ＆ドロップ
```

サーバー不要。ファイルをブラウザで直接開くだけで動作します。
